Follow this steps to test your endpoints

  • Run the program ("ExampleMvcApplication.java" file)
  • Open Postman
  • Create a GET request to the "login" endpoint (http://localhost:8080/login)
  • Send the following credentials in the request body :
    {
    "username" : "admin",
    "password" : "admin"
    }
    Use the JSON Format in the raw option. Don't copy the JSON we strongly suggest to type it directly in Postman.
  • The application will return a Bearer Token in the "Authorization" header of the response, save it or copy it wherever you want
  • Use the Bearer Token returned by the application in every endpoint you want to test

Good luck!
